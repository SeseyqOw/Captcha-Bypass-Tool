## ✨ Core Features

- **Multi-CAPTCHA Engine** — Cloudflare Turnstile • reCAPTCHA v2/v3/Enterprise • hCaptcha • GeeTest • DataDome • AWS WAF
- **Browser Fingerprint Emulation** — Chrome & Firefox profiles (canvas, WebGL, fonts, audio, hardware concurrency…)
- **IETF / Behavior Simulation** — Mouse curves, scroll patterns, keystroke dynamics, realistic timing
- **Proxy Support** — HTTP • SOCKS5 • Residential • Rotating lists
- **Parallel Bots** — Run multiple stealth instances with different fingerprints & proxies
- **Smart Delay Engine** — Human-like randomized delays + jitter
- **Github Sync** — Pull latest solver fingerprints & evasion configs
- **Igyx UI** — Clean dark theme with animated progress bars
- **Enesygurs Design System** — Modern, responsive controls

## 🚀 Quick Start

1. Launch the application
2. Configure desired features:

   - **Bypass** → Enable active solving engine
   - **IETF** → Advanced behavioral mimicry
   - **Chrome / Firefox** → Select fingerprint base
   - **Proxy** → Load list or enter single proxy
   - **Delay** → Human-like interaction timing
   - **Bots** → Number of concurrent sessions

3. Hit **Start** and watch the magic happen.
